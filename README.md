# Gatitos

## Objetivo /português

```
O webApp, Gatitos é uma das soluções disponíveis atualmente para coleta e adoção de animais de rua.
Abandono de cães aumentou em 60% no Rio desde 2020, aponta ONG Ajuda Patas, e para minímizar essa crescente, surgiu a ideia de atacar esse problema de forma descentralizada.
Rio das Ostras tem um número bastante elevado de abandono de animais e isso foi claramente evidenciado durante a pandemia. Minha intensão é divulgar esse webApp para que o objetivo de coletar e adotar animais de rua seja uma cultura em Rio das Ostras, e que essa cultura se espalhe pelo pais.
```

## objective /English

```
The Gatitos webapp is one of the solutions currently available for collecting and adopting stray animals.

The number of abandoned dogs has increased by 60% in Rio since 2020, according to the NGO Ajuda Patas, and to minimize this increase, the idea arose to tackle this problem in a decentralized manner.


Rio das Ostras has a very high number of abandoned animals and this was clearly evident during the pandemic. My intention is to promote this webapp so that the goal of collecting and adopting stray animals becomes a culture in Rio das Ostras, and that this culture spreads throughout the country..
```
